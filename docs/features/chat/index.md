# Chat

## Acerca de este índice

Contextura incluye un chat integrado con un asistente que conoce el contexto del proyecto abierto. El chat vive en el pane contextual derecho, junto al historial, y permite mantener conversaciones completas sin salir de la aplicación.

La implementación técnica vive en [../../chat.md](../../chat.md).

## Features

- [conversaciones.md](conversaciones.md): Creación, continuación y gestión de conversaciones con el asistente
  - Leer al tocar cómo se crean, guardan o retoman conversaciones

- [opciones.md](opciones.md): Controles para ajustar el modelo, el nivel de esfuerzo y el modo de respuesta
  - Leer al tocar las opciones de configuración del chat

- [comandos-rapidos.md](comandos-rapidos.md): Comandos rápidos con `/` para cambiar opciones o ejecutar acciones
  - Leer al tocar la entrada de texto del chat o los comandos disponibles

- [panel-contextual.md](panel-contextual.md): El pane derecho con pestañas para historial y chat
  - Leer al tocar la estructura del pane contextual o la alternancia entre pestañas
