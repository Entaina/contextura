# Modelo, esfuerzo y modo

## Qué hace

Permite ajustar tres parámetros del asistente directamente desde el chat: qué modelo usar, cuánto esfuerzo dedicar a la respuesta, y en qué modo operar.

## Experiencia del usuario

Debajo del campo de entrada del chat aparecen tres controles compactos (pastillas):

- **Modelo**: elige entre distintos modelos del asistente (más rápido, estándar o más capaz).
- **Esfuerzo**: controla cuánta reflexión dedica el asistente a cada respuesta (bajo, medio, alto, máximo).
- **Modo**: define cómo opera el asistente — responder preguntas sin hacer cambios, editar ficheros automáticamente, operar sin restricciones, o planificar sin actuar.

Al pulsar una pastilla se despliega la lista de opciones. La selección se aplica inmediatamente al siguiente mensaje y se recuerda entre sesiones.

Estos parámetros también se pueden cambiar con comandos rápidos (ver [comandos-rapidos.md](comandos-rapidos.md)).
