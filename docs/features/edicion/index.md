# Edición

## Acerca de este índice

Este módulo agrupa las features del área de edición: cómo se abren los ficheros, cómo se distribuyen en pestañas y paneles, el editor propiamente dicho y la pantalla inicial cuando no hay nada abierto. También cubre lo que la aplicación recuerda entre sesiones sobre la distribución y el estado de la edición.

La implementación técnica vive en [../../frontend.md](../../frontend.md).

## Features

- [tabs-splits.md](tabs-splits.md): Pestañas, splits, arrastre de pestañas y apertura en panel nuevo
  - Leer al tocar la distribución del área de edición

- [editor-wysiwyg.md](editor-wysiwyg.md): Editor de markdown con modos WYSIWYG y texto
  - Leer al tocar la experiencia de edición dentro de un panel

- [modo-editor-por-fichero.md](modo-editor-por-fichero.md): Memoria del modo (WYSIWYG o texto) por fichero
  - Leer al tocar la preferencia de modo del editor

- [drag-and-drop.md](drag-and-drop.md): Arrastrar un fichero del árbol al área de edición
  - Leer al tocar la apertura de ficheros por arrastre

- [welcome-watermark.md](welcome-watermark.md): Pantalla de bienvenida cuando no hay ninguna pestaña abierta
  - Leer al tocar el estado inicial del área de edición

- [layout-persistente.md](layout-persistente.md): Memoria del layout de pestañas y splits entre sesiones
  - Leer al tocar cómo Contextura recuerda la distribución al reabrir

- [ultimo-fichero.md](ultimo-fichero.md): Reapertura del último fichero al arrancar
  - Leer al tocar el estado inicial al abrir la app
